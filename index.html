A propos de React sur Linux

Installation : nodejs et npm version supérieur ou égale 14.

Au cas où on a une version de node qui n'est pas bon, on la supprime : sudo apt remove nodejs
Nouvelle version :
a°) curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
b°) sudo apt-get install -y nodejs
verification de version: node -v et npm -v
